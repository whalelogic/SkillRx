# UI/UX Development

Prompts, system instructions, and skills for UI/UX design and front-end development with Gemini AI.

## System Instructions

| File | Role Description |
|---|---|
| [`ux-designer.md`](system-instructions/ux-designer.md) | UX strategist covering research, information architecture, and interaction design |
| [`ui-developer.md`](system-instructions/ui-developer.md) | UI engineer bridging design and code — components, design systems, and accessibility |
| [`product-designer.md`](system-instructions/product-designer.md) | End-to-end product designer covering discovery through delivery |

## Prompts

| File | Task |
|---|---|
| [`design-review.md`](prompts/design-review.md) | Review a UI design for usability, accessibility, and visual hierarchy |
| [`component-generation.md`](prompts/component-generation.md) | Generate a UI component in HTML/CSS/React/Vue with accessibility built in |
| [`accessibility-audit.md`](prompts/accessibility-audit.md) | Audit a page or component against WCAG 2.2 AA criteria |
| [`user-flow-design.md`](prompts/user-flow-design.md) | Design or evaluate a user flow for a specific task |
| [`design-system-planning.md`](prompts/design-system-planning.md) | Plan or extend a design system with tokens, components, and documentation |

## Skills

| File | Capability |
|---|---|
