# Repository Analysis

Prompts, system instructions, and skills for codebase analysis, documentation generation, and reference creation with Gemini AI.

## System Instructions

| File | Role Description |
|---|---|
| [`codebase-analyzer.md`](system-instructions/codebase-analyzer.md) | Deep codebase analyst for architecture, patterns, and quality assessment |
| [`documentation-generator.md`](system-instructions/documentation-generator.md) | Specialist in extracting and generating documentation from code |
| [`dependency-auditor.md`](system-instructions/dependency-auditor.md) | Dependency and supply chain security auditor |

## Prompts

| File | Task |
|---|---|
| [`codebase-overview.md`](prompts/codebase-overview.md) | Generate a structured overview of an unfamiliar codebase |
| [`dependency-analysis.md`](prompts/dependency-analysis.md) | Analyse project dependencies for risk, outdatedness, and security |
| [`changelog-generation.md`](prompts/changelog-generation.md) | Generate a human-readable changelog from git history or PR descriptions |
| [`api-reference-generation.md`](prompts/api-reference-generation.md) | Generate a complete API reference from code or OpenAPI specs |
| [`technical-debt-assessment.md`](prompts/technical-debt-assessment.md) | Identify and prioritise technical debt in a codebase |
